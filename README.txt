Who Fed Yugi? - Final Design Project
Lillian Jones

This folder contains all necessary code to execute or re-create the project. This folder contains two sub-folders:

-"Main Application" - Folder containing all UI code, as well as back-end functionality (Python). Also contains spreadsheet to be read and updated as feeding log.

-"Hardware Code (Arduino IDE)" - Folder containing code uploaded to the ATMega328P. Main code is contained in "feeder_arduino", while all tests conducted for individual componenets are stored in the "Test" folder
